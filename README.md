# Credit Card Fraud Detection

## Project Overview

   Credit card fraud is a major concern in modern digital transactions, leading to significant financial losses for both customers and financial institutions. This project detects fraudulent transactions using Machine Learning and provides a Flask-based web interface for real-time prediction.

## Abstract

   This project identifies fraudulent transactions using ensemble Machine Learning algorithms such as Random Forest, AdaBoost, CatBoost, XGBoost, and LightGBM to improve prediction accuracy.

## Problem Statement

  * Fraud detection datasets are highly imbalanced
  * Lack of real-world datasets due to privacy concerns
  * Difficulty in distinguishing genuine and fraudulent transactions

## Features

  * Detects fraudulent transactions
  * Uses multiple Machine Learning algorithms
  * Handles imbalanced datasets
  * Real-time prediction using Flask

## Tech Stack

  * Python
  * Pandas, NumPy, Scikit-learn
  * Flask
  * Anaconda, Jupyter Notebook

## Workflow

  1. Data Collection
  2. Data Preprocessing
  3. Exploratory Data Analysis
  4. Train-Test Split
  5. Model Training (Random Forest, AdaBoost, CatBoost, XGBoost, LightGBM)
  6. Model Evaluation
  7. Prediction
## How to Run

   pip install -r requirements.txt
   python app.py

## Conclusion

   This project demonstrates how Machine Learning can effectively detect fraudulent transactions and reduce financial risks using real-time prediction.
